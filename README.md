# SELCO India — Lighting Lives, Powering Livelihoods

A research website built by **Pitch Perfects** (B.Tech IT, Ajeenkya DY Patil University) for the course *Entrepreneurship for Engineers (GEX209)*.

## About

This site presents our case study on **SELCO India's rural innovation model** — exploring how a user-centered ecosystem approach to clean energy can create sustainable livelihoods for underserved communities in rural India.

## Hosting on GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Under *Source*, select **GitHub Actions**
4. Push to `main` — the site will deploy automatically

Live URL will be: `https://<your-username>.github.io/<repo-name>/`
